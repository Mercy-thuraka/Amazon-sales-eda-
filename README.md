#  Amazon Sales EDA Project

##  Problem Statement
The objective of this project is to analyze Amazon sales data to identify trends, key revenue drivers, and customer behavior.

##  Objectives
- Analyze sales trends over time
- Identify top-performing products and categories
- Understand customer location impact
- Generate business insights

##  Data Cleaning
- Checked for missing values and duplicates
- Converted Date column to datetime

## Feature Engineering
- Extracted Month, Year, and Day from Date

##  Analysis Performed
- Monthly sales trend
- Category-wise analysis
- Top products
- Customer location analysis

 ##Key Insights
- Electronics dominates revenue
- Sales peak in Feb–March
- Revenue concentrated in few products
- Location impacts sales significantly

##  Conclusion
This project provides actionable insights to improve business decision-making.
